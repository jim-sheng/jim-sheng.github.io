不要放过core文件

问 ：Core文件是什么？
答 ：a disk file containing an image of the process’s memory at the time of termination


问 ：Core的作用？
答 ：
    1、让你在调试时，不用花费大量等待程序出错；
    2、让你避免了单步调试的烦恼
    3、让你定位错误所在
