etlua 用法

当有以下标签的时候 etlua 会将 Lua 的代码注入进模板：

  • <% lua_code %> 常规的 Lua 代码引入，类似 PHP 里面的 <?php xxxx ?>
  • <%= lua_expression %> 表达式输出结果，但是 HTML 会被转义
  • <%- lua_expression %> 和上面的功能一样，不过不会转义 HTML
